# hrpage

- **For starting**:
```bash
    $ ~ cd local
    $ ~ docker-compose up -d
```

- **Authentication page**
    - [http://127.0.0.1:5500/auth/](http://127.0.0.1:5500/auth/)

- **Other page**
    - [http://127.0.0.1:5500/page/](http://127.0.0.1:5500/page/)

- **For stopping**
```bash
    $ ~ cd local
    $ ~ docker-compose down
```
